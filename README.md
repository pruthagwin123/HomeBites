# HomeBites

Production-ready MERN starter structure for a startup-style home food delivery platform.

## Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express + MongoDB

## Product Modules
- Customer auth (signup/login)
- Chef auth (signup/login)
- Food listings
- Cart
- Order management and tracking
- Wallet
- Meal subscriptions
- Ratings and reviews
- Analytics dashboard
- OTP delivery verification flow
- Email notifications

## Folder Overview
```text
HomeBites/
  frontend/   # Vite + React + Tailwind application
  backend/    # Express + MongoDB API and background services
```

## Quick Start
1. Install dependencies
   - `cd frontend && npm install`
   - `cd ../backend && npm install`
2. Set environment variables
   - Copy `backend/.env.example` to `backend/.env`
3. Run apps
   - Frontend: `cd frontend && npm run dev`
   - Backend: `cd backend && npm run dev`
